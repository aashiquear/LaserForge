📖 Instructions ▼

🎮 Game Modes

  • Free Draw: Create anything!
  • 2D Shape: Match letters or symbols
  • 3D Shape: Build solid or frame objects

🎯 Controls

  • Draw: Touch/click directly on powder
  • Joystick: Move laser position
  • FIRE button: Toggle laser on/off
  • Camera: Buttons or pinch/drag

⚙️ Laser Shapes

  • Gaussian: Standard bell curve intensity
  • Top Hat: Uniform flat intensity
  • Doughnut: Ring pattern (hollow center)
  • Elliptical: Oval-shaped beam

🔧 Process

  • Watch recoater spread powder
  • Draw laser path (silver = melted metal)
  • Click "Next Layer" to continue
  • Click "Finish" when done
